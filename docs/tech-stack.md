# Tech Stack

## Runtime
- Chrome Extension (Manifest V3)
- WXT Framework (browser extension framework)

## Language
- TypeScript

## AI Integration
- OpenRouter API (single API key)
- Models: GPT-4o, Claude Sonnet 4.5, Gemini 2.5 Flash

## Storage
- chrome.storage.sync (API key, settings)
- chrome.storage.local (cached results)

## UI
- HTML + CSS (popup, no framework)

## Design Principle
- Copy-only output (no auto-populate into forms)
- Safe for all platforms (LinkedIn, Upwork, Fiverr, etc.)
